how to run project 
python -m http.server 8080
then open browser at localhost:8080

soundtrack Vivaldi-four seasons downloaded at https://www.classicals.de/vivaldi-seasons

3d model generated with https://hyper3d.ai/rodin
rigged and animated by mixamo https://www.mixamo.com

dragon model and animated from https://sketchfab.com/3d-models/tarisland-dragon-5cc18c454a83405984900ae1c654d7de

interactive camera insprited from https://github.com/icurtis1/off-axis-sneaker/tree/main
visual novel and storyteller narrative inspired from DnD

ai agents controlled by ollama models

most of the code is written with the support from Claude Sonnet 4.5

Some instructions:
- TAB for fast forwarding narrative.
- LEFT SHIFT for recording voice / RIGHT SHIFT for typing in conversation
- In battle:
 - Ronan - swordman: 1 - slash, 2 - heavy slash, 3 - block
 - Elric - archer: 1 - stab, 2 - shoot arrow, 3 - dodge
 - Seraphine - magician: 1 - cast spell, 2 - cast heavy spell, 3 - heal stamina
User can control the character by pressing these buttons during fighting, the other characters of the party and the boss are all controlled by AI agents
